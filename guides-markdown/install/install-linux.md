###### Source Install (Requires: Node >= 7.6)

```bash
git clone https://github.com/bcoin-org/bcoin.git
cd bcoin
npm install
bin/bcoin  #run
```

###### Archlinux AUR Package

```bash
yaourt -S bcoin-git
```
